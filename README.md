# Fase Final Mundial ADES

Sitio estático para apuestas de eliminación directa del Mundial 2026.

- `index.html`: aplicación principal para apostadores y administrador.
- `RESUMEN.md`: reglas, flujo de uso y notas técnicas.
- `ejemplo-apuesta-ficticia.html`: ejemplo de cómo se ve el archivo que descarga un apostador.

Publicado con GitHub Pages desde la rama `main`.
